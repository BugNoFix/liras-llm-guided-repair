
// Pattern name: SampleTransport

// Agents: Nurse, Robot

// Locations: PatientFloor, Laboratory

// Resources: Samples

// Conditions:

// - If the nurse is at the patient floor and there is a sample to be collected, then:

moveTo PatientFloor

// Wait for 5 minutes

// - If the robot is at the laboratory and there is a locked drawer available, then:

moveTo Laboratory

openLockedDrawer

// - If the nurse has inserted data about the sample in the track system, then:

insertDataInTrackSystem

// - If the time between sample collection and laboratory delivery is longer than 30 minutes, then:

waitFor30Minutes

// Actions:

// - Move to patient floor
moveTo PatientFloor

// - Collect sample using barcode scanning
scanBarcode

// - Move to laboratory
moveTo Laboratory

// - Open locked drawer
openLockedDrawer

// - Pick up sample using robotic arm or laboratory personnel
pickUpSample

// - Sort samples
sortSamples

// - Load samples into entry-module of analysis machines
loadSamples

// - Log information about where and when a given sample was picked securely
logSamplePickupInformation

// - Track time between sample collection and laboratory delivery
trackSampleDeliveryTime

// Sequences:

sequence:
1. moveTo PatientFloor

4. sortSamples

6. trackSampleDeliveryTime
